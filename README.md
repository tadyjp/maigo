## Installation


```ruby
gem 'maigo'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nokogiri
    $ gem install maigo

## Usage

Run `maigo` command:

    $ cd path/to/xcode-project-dir
    $ maigo

or

    $ maigo path/to/xcode-project-dir

and then, you will missing outlets information.
